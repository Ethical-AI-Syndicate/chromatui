# OpenAI/Anthropic Rider

This rider applies to this repository in addition to the MIT License in `LICENSE`.

## Purpose

This project allows use of AI-assisted workflows. The rider preserves provenance
and disclosure expectations for OpenAI/Anthropic-assisted modifications.

## Rider Terms

1. If a contributor uses OpenAI or Anthropic systems to generate substantial code,
   tests, documentation, or design artifacts submitted to this repository, the
   contributor should disclose AI assistance in the related commit message, PR,
   or release notes.
2. Distribution of modified versions should retain this rider file and the MIT
   license file.
3. Nothing in this rider grants trademark rights for OpenAI or Anthropic names,
   logos, or marks.

## Precedence

The MIT License remains the primary software license for this repository. This
rider adds project-level disclosure and provenance requirements.
