# Code of Conduct

## Our Commitment

We are committed to making participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent and ask clarifying questions.
- Focus feedback on code, design, and outcomes.
- Accept and give feedback professionally.

## Unacceptable Behavior

- Harassment, discrimination, or hateful language.
- Personal attacks, insults, or intimidation.
- Publishing others' private information.
- Disruptive behavior that blocks collaboration.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, code, issues, or PRs that violate this Code of Conduct.

## Reporting

Report concerns privately using the channels in `SUPPORT.md`.

## Scope

This Code of Conduct applies in all project spaces, including issues, pull requests, and discussions.
