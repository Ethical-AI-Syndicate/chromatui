# Support

## Getting Help

- Bug reports and feature requests: GitHub Issues
- Questions on usage and design: GitHub Discussions (if enabled) or Issues

When asking for help, include:

- Environment details (OS, terminal emulator, Rust version)
- Exact command(s) run
- Expected vs actual behavior
- Relevant logs or screenshots
